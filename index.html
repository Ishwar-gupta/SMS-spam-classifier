How the spam classifier works

The project is powered by NLP preprocessing, TF-IDF vectorization, and a Naive Bayes classifier trained on SMS and email data.

Preprocessing

  • Lowercase text to standardize input.
  • Remove punctuation and tokenize words.
  • Remove stopwords to focus on meaningful terms.
  • Apply stemming to unify similar words.

Vectorization

  • TF-IDF Vectorizer converts text to numerical features.
  • Top terms are selected to improve prediction speed.
  • Each message becomes a feature vector for the classifier.

Prediction

  • Multinomial Naive Bayes predicts spam or ham.
  • Fast, lightweight, and reliable for text classification.
  • Ideal for deployment as a web interface or app.
